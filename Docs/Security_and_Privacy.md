# Security and Privacy

## Principles
- Least privilege, data minimization, transparency.

## Controls
- JWT auth, role-based access.
- Encryption: TLS in transit; at-rest for sensitive fields.
- PII handling: hashed identifiers; configurable retention.

## Privacy
- Consent for analytics; opt-out settings.
- Access logs and audit trails.

## Compliance
- GDPR/FERPA-ready: data export, deletion, consent records.

## Guardrails
- Hallucination detection: citation requirement; confidence thresholds.
- Abuse prevention: rate limiting; content filters.

## Scaling
- Centralized secrets management; rotation policies.
- Automated vulnerability scanning.
